# 🍓 食いしん坊みくじ☆

**今日のメシ運うらない** — なまえをいれるだけで、今日のあなたのメシ運がわかります♡

👉 **占ってみる：https://shio-keio-edu.github.io/omikuji/**

## あそびかた

1. なまえ（ニックネームでOK）をいれる
2. 「はらぺこで引く♡」をタップ
3. きょうのメシ運が出ます！

- **メシ運ランク**：特盛吉☆／大盛吉／並盛吉／小盛吉／おかわり吉／腹八分
- **きょうの食べたい漢字**：麺・肉・甘・苺…など全24種
- **肉運・麺運・甘味運・揚げ運**をハート5段階で判定
- **開運メニュー＆魔法の調味料**つき

同じ名前なら、その日いちにち同じ結果☆
友達と名前をいれあって、メシ運くらべしてね。結果はXでシェアできます。

## しくみ・安心ポイント

- HTML1ファイルだけで動く、サーバーなしの静的サイトです
- **入力したなまえはどこにも保存・送信されません**（なまえ＋日付から計算で結果を出しているだけ）
- 占い結果はエンターテイメントです。食べすぎには注意してね🍰
